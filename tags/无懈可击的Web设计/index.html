使用em实现灵活的文字-无懈可击的Web设计

使用em单位也是一直替代关键字的字体设定策略,并且它是一个相对单位,且能够提供更高的精度和更好的控制.

em是一个可变的度量单位,1em等于一个字型尺寸, 6针字型中就是6个点的长度,12针字型中就是12点长度，因此在任何尺寸下em都是同比例的.

我网页中通常情况下当前字体默认的大小一般为16px,所以1em就等于16px,如果默认字体大小是11px,那么1em就是11px,而在设定字体行高、间距以及元素之间空隙时使用em的优势在于,如果字体大小发生改变,这些对象的尺寸也会按比例改变.
