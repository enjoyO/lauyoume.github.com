分享Typecho的HTML5/CSS3单栏风格皮肤

前一段时间曾更换了最新的HTML5/CSS3制作的皮肤，一直说着等完善之后会继续分享出来，可是前一段时间一直在忙别的事情，以致于都没怎么完善下皮肤，前一周好多人说起这个风格的皮肤很Beauty，于是我很欢喜，趁着周末，赶紧将皮肤完善了一下，马上就分享了出来，希望没让大家等急了！

这次比前一次的内容多多完善了一下，对于底部栏目增加了一些内容并让友情链接滚动了，其次内容页增加了上一篇和下一篇，并且更改了一些内容的显示CSS。也参考了@枫叶红秋雨和@指上漫步的一些设计。

皮肤名称命名为： whyyou，希望新的皮肤新的域名带来新的开始。

首页效果图

注意事项和使用插件

1. 需要修改/var/Widget/Comments/Archive.php，增加以下函数（添加到313行或其他你认为可行的地方）

public function hasPage(){
    return ($this->options->commentsPageBreak && $this->_total > $this->options->commentsPageSize);
}

2. 需要我修改的友情链接插件Links

3. 需要随机/热评文章插件ArticleList

4. 随机热评文章个数和文字个数请在插件激活后设计

4. 需要Smilies表情插件
