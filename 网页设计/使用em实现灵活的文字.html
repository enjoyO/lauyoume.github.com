使用em实现灵活的文字-无懈可击的Web设计

使用em单位也是一直替代关键字的字体设定策略,并且它是一个相对单位,且能够提供更高的精度和更好的控制.

em是一个可变的度量单位,1em等于一个字型尺寸, 6针字型中就是6个点的长度,12针字型中就是12点长度，因此在任何尺寸下em都是同比例的.

我网页中通常情况下当前字体默认的大小一般为16px,所以1em就等于16px,如果默认字体大小是11px,那么1em就是11px,而在设定字体行高、间距以及元素之间空隙时使用em的优势在于,如果字体大小发生改变,这些对象的尺寸也会按比例改变.

文中引用了.Richard Rutter的一个巧妙方法来用em设置字体大小的基准值《文章地址》，其中的单位其实与像素等价，它是假定浏览器默认的字体大小为中，大多数情况这个值为16px，因此这个基准值的方法其实是

body { font-size:62.5%;}

这个神奇的百分比就是将网页的字体的基准值设置为10px，以后的处理字体大小都是很方便的整数，并且在使用em设置时完全可以再次考虑像素.

因此在这种情况下，我们要为各个元素设定不同的字体的值的css代码应该如下：

body { font-size:62.5%;} /* 基准值 10px */
h1 { font-size: 2em; } /* 20px */
h2 {font-size: 1.8em; } /* 18px */
P {font-size: 1.2em; } /* 12px */
#sidebar { font-size: 1.4em;} /* 14px */

还有特殊的情况，则是字体大小嵌套的下像素值问题。为了将p下的元素字体大小设定为24px，通常情况下我们会写的可能是：
P {font-size: 1.2em; } / 12px /
p abbr { font-size: 2.4em;}

但实际上的字体大小会变成28.8px,下面有一个非常简单的公式：
child pixels ÷ parent pixels = child ems

即只要套入上面的公式

24 px ÷ 12 px = 2  ems

那么实际上将段落中的abbr元素设定为24px的代码应该如下:

p abbr { font-size: 2em;}  /* 24px */

对此，其实我们只要谨记一点：子元素的字体大小如果指定了em 一定跟父节点的字体大小相关，即是子元素的em值乘以父元素的的字体大小！.
