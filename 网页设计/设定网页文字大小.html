使用关键字和百分比来设定文字大小-无懈可击的Web设计

通常情况下，在为网站页面设计文字大小的时候，我通常会以像素值来设定文字大小，因为这样精准的大小，在几乎每个浏览器上都是一样的大小的，这样基本能够让我精准的控制它们的显示尺寸。以前我是非常喜欢这么做的，但是自从看了该书之后，让我豁然明白，居然还能有这么好的方案，可以非常灵活的控制页面的字体。

我们知道，在css中主要有4大类别来设定font的fontsize:

1. 长度单位

1.1 相对长度

  • em：相对父级元素的字体大小
  • ex：相对特定字体中字母x的高度
  • px：相对于特定设备的分辨率

1.2 绝对长度单位

  • in：英才
  • cm：厘米
  • mm：毫米
  • pt：点
  • pc：12点活字

2. 相对大小的关键字

larger，smaller，这两个都是跟父级的子字体相关的.

3. 百分比值

如120%，在父级元素基础上增大20%，父级10px,那么120%则是12px

4. 绝对大小的关键字

xx-small,x-small,small,medium,large,x-large,xx-large. 这些关键字对应的字体大小则由浏览器或设备决定的.

在这些认知上，书中为我们给出了两种策略，如下我将我所看到的分享出来，这些让我明白了如何应用这些策略来定制网站页面的字体，并让我将这种策略应用到网页设计了，目前我应用em的字体策略并重新架构设计了搜街宝手机网站页面。

一. 使用关键字和百分比

使用关键字和百分比，一般情况下要给页面设定一个基准值，这个基础值使用small关键字.

body{
    font-size:small;
}

关键字设定的字体大小与当前浏览器的设置相关,.至于为什么选择small,通常情况下,small关键字默认的字体大小跟12px像素级别的字体大小很是接近.
当我们有了这个基准值之后,我们可以使用百分比来增大或减小字体大小，来获得我们想要的尺寸.

h1{
    font-size:150%;
}
.note{
    font-size:90%;
}

之后,我们结合使用关键字和百分比就可以获取我们的设计,它使得我们方便于管理整个页面的字体,亦方便于修改.

此外,在结合使用关键字和百分比时,基于各种设计相关的理由通常在基准值之上使用一个div来设定能够影响整个页面的百分比,如下：

body{
    font-size:small;
}
.wrap{
     font-size:95%;
}
h1{
    font-size:150%;
}
<body>
    <div class="wrap">
      <h1>标题</h1>
    </div>
</body>

另外，要谨慎使用嵌套的百分比，如上例子. h1其实是在95%的基础上再扩大至120%.. 即 默认字体大小 95% 150%.
